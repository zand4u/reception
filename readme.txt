# Reception App

Detta är en fullstack-applikation som hanterar registreringsnummer. 

## Backend

Backend är byggt med Node.js och Express och ansluter till en PostgreSQL-databas.

## Frontend

Frontend är byggt med HTML, CSS och JavaScript.

## Installation

1. Klona projektet.
2. Installera nödvändiga beroenden i både backend och frontend.
3. Starta backend med `npm run dev` och öppna frontend genom att öppna `index.html` i din webbläsare.
